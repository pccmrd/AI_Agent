# Твой личный ИИ-агент

## Описание проекта

В этом проекте был разработан персональный ИИ-агент, который помогает в образовательном пути и выборе карьеры. Агент реализован в виде Telegram-бота с возможностями:

- Помощи в выборе направления обучения
- Отправки регулярного дайджеста новых материалов по выбранному профилю
- Анализа рынка вакансий и отслеживания новых требований и компетенций
- Помощи в подборе актуальных материалов для выполнения учебных проектов

Проект полностью реализован с использованием инструментов n8n и облачных сервисов Cloud.ru, включая Foundation Models для работы с GigaChat API.

## Технологии и инструменты

- **n8n** - платформа для автоматизации рабочих процессов без написания кода
- **Cloud.ru Container Apps** - сервис для запуска контейнерных приложений
- **Cloud.ru Foundation Models** - сервис для доступа к API фундаментальных моделей машинного обучения
- **Cloud.ru Object Storage** - объектное хранилище для постоянного хранения данных
- **GigaChat** - российская языковая модель для генерации текста и анализа данных
- **Telegram Bot API** - для создания пользовательского интерфейса бота
- **Tavily Search API** - для поиска актуальной информации в интернете
- **RSS-агрегаторы** - для сбора новостей из Telegram-каналов

## Архитектура решения


Архитектура проекта построена на основе микросервисного подхода с использованием n8n в качестве центрального оркестратора. Telegram-бот выступает в качестве пользовательского интерфейса, а все логика обработки реализована через воркфлоу n8n с подключением различных AI-моделей и внешних сервисов.

## Реализованные функции

### Задание 1. Знакомство с инструментами

В рамках первого задания была настроена инфраструктура для работы проекта:

- Развернут n8n в облачной платформе Cloud.ru с использованием сервиса Container Apps
- Настроено постоянное хранилище для сохранения данных между перезапусками контейнера
- Подключены AI-модели через сервис Foundation Models Cloud.ru
- Созданы необходимые API-ключи для работы с внешними сервисами
- Реализован базовый воркфлоу для взаимодействия с моделью GigaChat


**Результат:** в папке `task1` размещен скриншот, демонстрирующий работу с моделью GigaChat через n8n.

### Задание 2. Твой первый ИИ-агент. Выбор специализации

Реализован Telegram-бот, выступающий в роли карьерного консультанта:

- Создан Telegram-бот через @BotFather с уникальным токеном
- Разработан системный промпт для AI-агента, задающий цель и правила поведения
- Подключен компонент SimpleMemory для хранения контекста диалога с каждым пользователем
- Реализовано задание серии вопросов для анализа:
  - Предыдущий жизненный опыт и достижения
  - Опыт прохождения отборочного бассейна
  - Личные предпочтения и интересы
  - Цели обучения и желаемая скорость их достижения
  - Сильные и слабые стороны
  - Предпочтения по формату работы
- Настроена генерация персональных рекомендаций по выбору специализации

Все персональные данные перед отправкой в AI-модели проходят процедуру анонимизации для обеспечения безопасности.


**Результат:** в папке `task2` размещены:
- `career_advisor_workflow.json` - экспортированный воркфлоу n8n
- `career_advisor_demo.pdf` - скриншоты финального взаимодействия с ботом
- `system_prompt_analysis.txt` - анализ ключевых элементов системного промпта

### Задание 3. Дайджест новостей

Реализована автоматическая система сбора и анализа новостей по выбранной специализации:

- Создан приватный Telegram-канал для получения дайджестов
- Подобраны 3+ Telegram-канала по рекомендованной специализации
- Настроен парсинг каналов через RSS-агрегатор FetchRSS
- Разработан воркфлоу для ежедневного формирования дайджеста:
  - Сбор новых постов за сутки
  - Анализ и краткое резюмирование каждого поста с помощью GigaChat-2
  - Формирование общего саммари по всем постам
  - Структурирование информации в удобном формате
  - Отправка дайджеста в приватный канал

Формат дайджеста включает:
1. Общее саммари о постах в одном абзаце
2. Группировку по источникам с детализацией по каждому посту
3. Ссылки на оригинальные материалы


**Результат:** в папке `task3` размещен файл `news_digest_workflow.json` с экспортированным воркфлоу и `news_digest_demo.pdf` с примером работы.

### Задание 4. Анализ вакансий

Реализована система анализа рынка труда для отслеживания востребованных навыков:

- Настроен парсинг вакансий с hh.ru через API
- Разработан воркфлоу для еженедельного сбора данных
- Реализован алгоритм нормализации навыков (приведение к единому виду)
- Создан AI-агент на базе GigaChat-2 для:
  - Извлечения ключевых требований из описаний вакансий
  - Определения частоты упоминаний каждого навыка
  - Формирования рейтинга востребованных компетенций
- Интеграция с Telegram-каналом для отправки результатов анализа

Для обеспечения безопасности и экономии ресурсов:
- Ограничен объем анализируемых данных при отладке
- Реализован контроль качества анализируемых данных
- Внедрены механизмы проверки корректности выводов AI



**Результат:** в папке `task4` размещены `vacancy_analysis_workflow.json` и `vacancy_analysis_demo.pdf` с результатами работы.

### Задание 5. Персональный ментор

Создана система персонального обучения и менторства:

- Реализован диалог для определения направления обучения
- Разработан алгоритм формирования учебного плана на основе запроса пользователя
- Интегрирован Tavily Search API для поиска актуальных материалов
- Создан механизм отслеживания прогресса обучения через Static Data в n8n
- Реализована система неповторяющихся постов по каждому изучаемому понятию
- Добавлена логика циклического обучения с возможностью выбора новых тем

Бот помогает в изучении как базовых концепций, так и углубленных тем в рамках подготовки к собеседованиям и выполнению учебных проектов.


**Результат:** в папке `task5` размещены `personal_mentor_workflow.json` и `personal_mentor_demo.pdf` с демонстрацией работы системы.

## Установка и настройка

Для развертывания проекта необходимо:

1. **Подготовка облачной инфраструктуры:**
   - Зарегистрироваться в Cloud.ru
   - Создать бакет в Object Storage
   - Настроить Container Apps для развертывания n8n
   - Получить API-ключи для Foundation Models

2. **Настройка n8n:**
   - Импортировать воркфлоу из папок `task2`, `task3`, `task4`, `task5`
   - Заполнить credentials для всех сервисов (Telegram, GigaChat, Tavily Search)
   - Настроить расписание для автоматических задач (дайджесты, анализ вакансий)

3. **Настройка Telegram-бота:**
   - Зарегистрировать бота через @BotFather
   - Создать приватный канал для дайджестов
   - Подключить бота к каналу с правами администратора

4. **Настройка дополнительных сервисов:**
   - Зарегистрироваться в FetchRSS для создания RSS из Telegram-каналов
   - Получить API-ключ в Tavily Search для поиска материалов

Детальная инструкция по настройке находится в файле [SETUP.md](SETUP.md).

## Использование

После настройки всех компонентов:

1. Запустите Telegram-бота, перейдя по ссылке: [@ваш_бот](https://t.me/ваш_бот)
2. Следуйте инструкциям бота для получения рекомендаций по выбору специализации
3. Для получения дайджеста новостей подпишитесь на ваш приватный канал
4. Дайджесты будут приходить автоматически по расписанию
5. Для получения персонального плана обучения воспользуйтесь командой `/learn` в боте

Команды бота:
- `/start` - начало работы с ботом
- `/specialization` - запуск процесса выбора специализации
- `/news` - получение последнего дайджеста новостей
- `/vacancies` - получение анализа вакансий за текущую неделю
- `/learn` - запуск персонального плана обучения
- `/help` - справка по командам
